# Мобильное приложение Lyra — требования

## Назначение

Аутентификация пользователей 1С через QR-код. Мобильное приложение — единственный источник userId для системы. По userId идёт биллинг.

## Сценарий аутентификации

1. Пользователь открывает обработку в 1С
2. 1С подключается к Bridge, получает QR-код
3. 1С показывает QR на форме: "Отсканируйте QR-код в приложении Lyra"
4. Пользователь открывает мобильное приложение, сканирует QR
5. Приложение отправляет подтверждение в Bridge (с userId)
6. Bridge проверяет баланс, разрешает или отказывает в подключении
7. 1С получает ответ и начинает работу (или показывает "пополните баланс")

## Функции приложения

### MVP

- Регистрация / вход (userId создаётся при регистрации)
- Сканер QR-кода
- Подтверждение подключения к сессии 1С
- Просмотр баланса
- Пополнение баланса

### Позже
