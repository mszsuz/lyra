## Синтаксис запросов 1С

Язык запросов 1С — НЕ SQL. Ключевые слова на русском.

### Даты в параметрах запросов

Для передачи дат используй параметр `date_params` — массив имён параметров, которые являются датами. Формат значения — ISO (`"2026-03-15"` или `"2026-03-15T00:00:00"`).

Пример — остатки на дату:
- `params: {"Дата": "2026-03-15"}`, `date_params: ["Дата"]`

**НЕ НУЖНО** писать `ДАТАВРЕМЯ()` в тексте запроса — используй `&Параметр` + `date_params`.

**Важно:** дата в виртуальных таблицах — граница «строго до». `Остатки(&Дата)` вернёт остатки до указанного момента, не включая его. Для остатков «на сегодня» (включая движения сегодня) передавай завтрашнюю дату.

### Виртуальные таблицы регистров накопления

```
-- Остатки на текущий момент
ВЫБРАТЬ * ИЗ РегистрНакопления.ИмяРегистра.Остатки

-- Остатки на дату (date_params: ["Дата"])
ВЫБРАТЬ * ИЗ РегистрНакопления.ИмяРегистра.Остатки(&Дата, )

-- Обороты за период (date_params: ["Начало", "Конец"])
ВЫБРАТЬ * ИЗ РегистрНакопления.ИмяРегистра.Обороты(&Начало, &Конец, , )
```

### Регистры бухгалтерии

```
-- Остатки по счёту (date_params: ["Дата"])
ВЫБРАТЬ * ИЗ РегистрБухгалтерии.Хозрасчетный.Остатки(&Дата, Счет = &Счет, )

-- Обороты (date_params: ["Начало", "Конец"])
ВЫБРАТЬ * ИЗ РегистрБухгалтерии.Хозрасчетный.ОборотыДтКт(&Начало, &Конец, , СчетДт = &Счет, , )
```

### Основные конструкции

```
ГДЕ Поле = &Значение
ГДЕ НЕ ПометкаУдаления
УПОРЯДОЧИТЬ ПО Поле ВОЗР
СГРУППИРОВАТЬ ПО Поле1, Поле2
СУММА(Поле), КОЛИЧЕСТВО(*), МАКСИМУМ(Поле)
ПЕРВЫЕ 20
```

### Типичные ошибки

- Угадать название таблицы → ошибка. Сначала lyra_meta_list!
- InformationRegisters для остатков → неправильно. Остатки — в AccumulationRegisters
- Многострочный код в lyra_data_eval → ошибка. Только одиночные выражения
- Результаты запросов ограничены 100 строк — используй ПЕРВЫЕ N, СГРУППИРОВАТЬ ПО или сузь условия ГДЕ
- `ДАТАВРЕМЯ()` в тексте запроса — устаревший способ. Используй `&Параметр` + `date_params`
- Запрос остатков без даты → включает будущие движения, результат неверный. Всегда передавай дату. «Покажи остатки» = завтрашняя дата в `date_params`
