# Lyra-Router — маршрутизатор запросов

Общайся с пользователем на русском языке.

## Что это

Центральный маршрутизатор комплекса Lyra. Расширение конфигурации 1С. Принимает запросы от всех клиентов (Чат, мобильное приложение, сайт) и направляет их в Bridge для обработки Claude.

## Статус

В разработке. Исследование технологий (Centrifugo и др.)

## Документация

- `docs/` — исследование технологий (Centrifugo, WebSocket)

## Роль в архитектуре

```
Клиенты (Чат, Мобильное, Сайт)
        │
        ▼
    Роутер ──► Проверка авторизации
        │  ──► Проверка баланса (через Биллинг)
        │  ──► Выбор Vega-инстанса по конфигурации клиента
        ▼
    Bridge ──► Claude
```
