# Топ дебиторов (задолженность покупателей)

## Источник в конфигурации

Функция `ЗадолженностьПоКонтрагенту` (Справочники.Контрагенты, ManagerModule) — готовый запрос по задолженности, авансам и неоплаченным счетам.

## Ключевые находки

- **Таблица:** `РегистрБухгалтерии.Хозрасчетный.Остатки`
- **Счета покупателей (62):** `РасчетыСПокупателями`, `РасчетыСПокупателямиВал`, `РасчетыСПокупателямиУЕ`
- **Счета поставщиков (60):** `РасчетыСПоставщиками` и аналогичные (Вал, УЕ)
- **Субконто:** Контрагенты (Субконто1), Договоры (Субконто2)
- **Дебиторка покупателей** = `СуммаРазвернутыйОстатокДт` по счёту 62
- **Кредиторка поставщикам** = `СуммаРазвернутыйОстатокКт` по счёту 60
- **Вид субконто:** `ЗНАЧЕНИЕ(ПланВидовХарактеристик.ВидыСубконтоХозрасчетные.Контрагенты)`

## Готовый запрос — топ N дебиторов

```bsl
ВЫБРАТЬ ПЕРВЫЕ 5
    Остатки.Субконто1 КАК Контрагент,
    СУММА(Остатки.СуммаРазвернутыйОстатокДт) КАК Задолженность
ИЗ
    РегистрБухгалтерии.Хозрасчетный.Остатки(
        ,
        Счет В (ЗНАЧЕНИЕ(ПланСчетов.Хозрасчетный.РасчетыСПокупателями), ЗНАЧЕНИЕ(ПланСчетов.Хозрасчетный.РасчетыСПокупателямиВал), ЗНАЧЕНИЕ(ПланСчетов.Хозрасчетный.РасчетыСПокупателямиУЕ)),
        ЗНАЧЕНИЕ(ПланВидовХарактеристик.ВидыСубконтоХозрасчетные.Контрагенты),
        ) КАК Остатки
СГРУППИРОВАТЬ ПО
    Остатки.Субконто1
УПОРЯДОЧИТЬ ПО
    Задолженность УБЫВ
```

## Аналогичный запрос — топ кредиторов (задолженность поставщикам)

Заменить счета на `РасчетыСПоставщиками` (60) и поле на `СуммаРазвернутыйОстатокКт`.
