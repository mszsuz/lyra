# Остатки на расчётных счетах (счёт 51)

## ⚠️ ВАЖНО: дата в виртуальной таблице Остатки

Из документации 1С: **«Если параметр не указан, то будут получены текущие остатки (на максимальную дату движений регистра)»**. Это НЕ текущая дата — если в базе есть документы из будущего, результат будет неверным.

**Также:** если передана Дата — остатки на **начало дня** (движения этого дня НЕ учтены). Для остатков «на сегодня» (включая сегодняшние движения) передавать **завтрашнюю дату**.

**Правило:** «покажи остатки» = остатки на сейчас. Молча подставляй `ДАТАВРЕМЯ(год, месяц, день+1)` от текущей даты (завтра, чтобы учесть движения сегодня). Если пользователь указал конкретную дату — используй её (+ 1 день если нужны остатки на конец указанного дня).

## Запрос — остатки в разрезе банковских счетов

```bsl
ВЫБРАТЬ
    Остатки.Организация.Наименование КАК Организация,
    Остатки.Субконто1 КАК БанковскийСчет,
    Остатки.СуммаОстатокДт КАК Остаток
ИЗ
    РегистрБухгалтерии.Хозрасчетный.Остатки(
        ДАТАВРЕМЯ(2026, 3, 15),
        Счет.Код = "51",
        ЗНАЧЕНИЕ(ПланВидовХарактеристик.ВидыСубконтоХозрасчетные.БанковскиеСчета),
        ) КАК Остатки
ГДЕ
    Остатки.СуммаОстатокДт <> 0
УПОРЯДОЧИТЬ ПО
    Организация
```

## Запрос — остатки итого по организациям (без субконто)

```bsl
ВЫБРАТЬ
    Остатки.Организация.Наименование КАК Организация,
    Остатки.Счет.Код КАК КодСчета,
    Остатки.СуммаОстатокДт КАК Остаток
ИЗ
    РегистрБухгалтерии.Хозрасчетный.Остатки(ДАТАВРЕМЯ(2026, 3, 15), , , ) КАК Остатки
ГДЕ
    Остатки.Счет.Код = "51"
    И Остатки.СуммаОстатокДт <> 0
УПОРЯДОЧИТЬ ПО
    Организация
```

**Примечание:** запрос без субконто использует фильтр по счёту в ГДЕ, а не в параметрах виртуальной таблицы.

## Ключевые моменты

- **Третий параметр** виртуальной таблицы — вид субконто: `ЗНАЧЕНИЕ(ПланВидовХарактеристик.ВидыСубконтоХозрасчетные.БанковскиеСчета)`. Без него Субконто1 недоступно
- **Дата** передаётся через `ДАТАВРЕМЯ()` в тексте запроса (строка через параметр не приводится к дате)
- Пустой банковский счёт (id = 00000000-...) — документы без заполненного субконто

## Смежные счета

- **50.01** — Касса организации
- **52** — Валютные счета
- **57.03** — Продажи по платежным картам
