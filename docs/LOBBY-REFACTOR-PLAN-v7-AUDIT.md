## Проверка внутренней согласованности

### Инварианты (7 шт.) vs алгоритмы

Все 7 инвариантов подтверждаются соответствующими алгоритмами и dispatch-логикой. Противоречий нет.

### Этапы vs зависимости

Цепочка зависимостей корректна:
- Этап 0 → ни от чего
- Этап 1 → Этап 0
- Этап 2 → Этап 0 + 1
- Этап 3 → Этап 1
- Этап 4 → все

### Definition of Done (10 пунктов)

Каждый пункт покрыт тестами из Этапов 0–2 или прямо следует из инвариантов и алгоритмов.

## Замечания

Замечаний нет. План готов к реализации.

Единственное наблюдение, не требующее действий: параметр `clientUser` (sub из lobby JWT) убран из сигнатуры `bootstrapChat`/`bootstrapMobile` в v7, но всё ещё нужен для `apiSubscribe(user, client, channel)`. При реализации его нужно будет извлечь из join event info. Это не ошибка плана — просто деталь реализации.
